# AI Data Pipeline & EDA Project

This project provides an automated pipeline for cleaning, integrating, and analyzing sales data. Follow the instructions below to set up the environment and run the analysis.


### 1. Environment Setup 
Run the following command to install all required libraries:
```bash
uv sync
```

### 2. Running the Analysis
To view the Exploratory Data Analysis (EDA) and run the full pipeline, use the following command to open the Jupyter notebook:
```bash
uv run jupyter lab notebook/eda.ipynb
```
